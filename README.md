# Squeaky App!

**NOTE**: Squeaky was sunsetted at the end of May 2024.

![Squeaky dashboard](./docs/dashboard.jpg)

### Requirements
- Node.js v16

### Installation
```shell
$ git clone git@github.com:squeaky-ai/app.git
$ cd squeaky
$ yarn install
```

### Fetching the latest GraphQL types
```shell
$ yarn codegen
```

### Run the server
```shell
$ yarn dev
```
